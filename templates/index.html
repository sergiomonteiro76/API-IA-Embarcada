📚 Exemplos de Uso da API

Exemplos de como chamar a API usando diferentes ferramentas

🌐 JavaScript (Fetch API)

// Análise de Sentimento
fetch('http://localhost:5000/api/sentimento', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        texto: 'Estou muito feliz hoje!'
    })
})
.then(response => response.json())
.then(data => console.log(data));

🐍 Python (Requests)

import requests

# Gerar Texto
url = 'http://localhost:5000/api/gerar'
dados = {
    'tema': 'Inteligência Artificial',
    'tamanho': 'medio'
}
resposta = requests.post(url, json=dados)
print(resposta.json())

💻 cURL

# Resumir Texto
curl -X POST http://localhost:5000/api/resumir \
  -H "Content-Type: application/json" \
  -d '{
    "texto": "Seu texto aqui...",
    "tamanho_resumo": "medio"
  }'

POST /api/gerar

POST /api/resumir
